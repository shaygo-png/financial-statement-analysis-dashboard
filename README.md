# Apple Financial Statement Analysis Dashboard

This project analyzes the financial performance of Apple Inc. from 2021–2023 using Microsoft Excel. The goal was to evaluate key financial metrics and build a dashboard that visualizes company performance.

## Project Overview

The project uses publicly available financial statement data from Apple Inc. to calculate financial ratios and visualize trends in company performance.

## Data Used

Financial statement data includes:

- Revenue  
- Net Income  
- Total Assets  
- Total Liabilities  

## Financial Ratios Calculated

The following ratios were calculated in Excel:

- Profit Margin  
- Return on Assets (ROA)  
- Debt Ratio  

## Dashboard Visualizations

The Excel dashboard includes charts for:

- Revenue Trend (2021–2023)  
- Net Income Trend (2021–2023)  
- Profit Margin Trend  

These visualizations help evaluate Apple's profitability and financial performance over time.

## Skills Demonstrated

- Financial statement analysis  
- Ratio analysis  
- Microsoft Excel dashboard creation  
- Data visualization  

## How to Use

1. Download the Excel file below.  
2. Open the workbook in Microsoft Excel.  
3. View the **Dashboard** sheet to see the financial analysis visualizations.
